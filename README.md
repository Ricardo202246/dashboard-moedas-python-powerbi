# 📊 Dashboard de Moedas com Python + Power BI

Pipeline de dados desenvolvido para coleta, tratamento e análise histórica de moedas utilizando Python, API REST e Power BI.

---

# 🚀 Objetivo do Projeto

O objetivo deste projeto foi construir uma solução para acompanhamento histórico de moedas, permitindo visualizar tendências, variações e análises temporais através de dashboards interativos.

O projeto realiza:
- Coleta automática de dados via API
- Tratamento e transformação dos dados com Python
- Armazenamento histórico das cotações
- Construção de dashboard interativo no Power BI

---

# 🛠️ Tecnologias Utilizadas

- Python

- Power BI
- API REST
